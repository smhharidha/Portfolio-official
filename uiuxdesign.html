Amazon HomePage

This UI/UX project presents a mobile mockup of the Amazon shopping app, designed using Figma. It replicates a typical e-commerce experience with sections for search, categories, offers, and payments, following key UI/UX principles like visual hierarchy and consistency. Tools like Frames, Components, and Prototyping were used to structure screens, create reusable elements, and enable interactive navigation. Realistic mock content such as prices and QR scan features enhance user simulation, making it suitable for usability testing. The design focuses on a mobile-first, responsive approach, aligning with modern digital commerce needs.

Coffee Shop App

This UI/UX project showcases a coffee ordering mobile app prototype, designed using Figma. The design includes a smooth user flow starting from the welcome screen to the order confirmation page. Key UX principles applied are intuitive navigation, clear content categorization, and emotional design—evident from warm color tones, appealing product images, and encouraging messages like “Order placed successfully!” with emoji. UI elements such as cards, tabs, and icons enhance usability and ensure a pleasant user experience. Tools used include Figma’s Frame tool for each iPhone screen, Components for reusable buttons and icons, and Prototyping features to simulate user interactions like selecting coffee, customizing quantity, and placing an order. The app design follows a dark theme, aligning with coffee aesthetics, and targets quick, user-friendly ordering for coffee lovers.
