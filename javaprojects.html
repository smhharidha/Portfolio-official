Pharmacy Management System
  • The Pharmacy Management System is a Java-based console application that helps manage medicine inventory and sales in a pharmacy.
  • It allows users to add new medicines with details like name, category, stock, and price, update or remove existing medicines, view all available medicines, and record sales by deducting quantities from stock.
  • The system keeps a history of all sales made, enabling users to review past transactions.
  • It uses classes like Medicine to represent individual items and PharmacyManagementSystem to handle operations using data structures such as HashMap for inventory and ArrayList for sales.
