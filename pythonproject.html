Python Project

Medium article reader
  • This Python script takes a URL as input, fetches the article content from the webpage using requests and BeautifulSoup, extracts all paragraph text, and then reads it aloud using the pyttsx3 text-to-speech engine.
  • It also saves the spoken content as an audio file (output.wav) on the user's desktop.
  • This tool combines web scraping with voice output, making it super helpful for auditory learners, accessibility tools, or even hands-free reading experiences.
